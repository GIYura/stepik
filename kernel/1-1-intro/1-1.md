Для разработки модулей ядра Linux необходимо установить:
- build-essential (gcc/make/gdb ...);
- vim;
- git;
- libncurses5-dev;
- bc (calculator);

**Example:**
```
apt-get install build-essential vim git libncurses5-dev bc
```
